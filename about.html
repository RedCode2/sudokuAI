About Sudoku AI

Introduction

This project is made for solving Sudoku Puzzles. It can Solve a Sudoku Puzzle for you, or generate a Sudoku Puzzle for you to solve. This project is open source. View the source of this project by clicking on this link.

How to use the Sudoku Solver?

  1. Enter your sudoku board in each cell of the board.
  2. Click on the Solve Sudoku button present at the bottom right panel of the Sudoku Solver.
  3. The Sudoku Puzzle will get solved in under a second and will be displayed on the board. However, the sudoku won't solve if the question itself is invalid. So check your question first before using the Sudoku Solver.

How to use the Sudoku Solver Interface?

  • To change window settings, use the following component present at the Sudoku Solver Sidebar. It's recommended to not change any of these settings as the default look is optimized for the Sudoku Solver.
  • To change the appearance of the Sudoku Board, use the following component present at the Sudoku Solver Sidebar. These settings are switched to default after reloading the page.

Command List

  • pg_ff default: Changes the font family of the page to default.
  • pg_ff sans: Changes the font family of the page to sans-serif.
  • pg_ff serif: Changes the font family of the page to serif.
  • pg_ff mono: Changes the font family of the page to monospace.
  • pg_fs small: Changes the font size of the page to small (Default & Recommended).
  • pg_fs medium: Changes the font size of the page to medium.
  • pg_fs large: Changes the font size of the page to large.
  • solve_sudoku: Solves the Sudoku Board.
  • clear_board: Clears the Sudoku Board.
